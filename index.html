Venn

Single-page, full-stack application inspired by Facebook. Venn is a social media platform that features React on the frontend with Redux for state management. The backend, which is built on Rails, is a RESTful API that serves up JSON data. Venn also integrates AWS for photo storage.

Maze Runner

Tool for visualizing maze generation and path finding algorithms built with JavaScript, jQuery and HTML5 Canvas. Featured search algorithms include Breadth-First Search (BFS), Depth-First Search (DFS), Dijkstra's Algorithm and A*. Maze Runner incorporates Prim's algorithm for random maze generation.

DOM Querylone

I'm going to make you an offer you can't refuse. With DOM Querylone, you pull all the strings. Inspired by jQuery, DOM Querylone is built with vanilla JavaScript and gives you the power to traverse the DOM, manipulate elements, leverage event handlers and even make AJAX requests. Checkout my simple snake game to get a glimpse of what you can do with DOM Querylone.
